# Glossary

- Graphiti: Temporal knowledge graph layer.
- FalkorDB: Graph database (Neo4j-compatible) used for storage.
- GraphRAG: Semantic discovery layer over unstructured evidence.
- Ralph Loop: Multi-pass validation system for signals.
- Signal: Structured, validated unit of intelligence for an entity.
- Episode: Temporal event in the graph.
- MCP: Model Context Protocol tool servers.
- CopilotKit: UI runtime for chat + tool invocation.
